## 🦀 Rust `loop` 実装ドリル

### 第1問：数当てゲームの「入力待ちループ」

ユーザーが特定の数字を当てるまで、何度も入力を促すループを実装してください。

* 変数 `target` に正解の数字（例: `7`）を入れておきます。
* `loop` を使い、毎回「数字を入力してください」と表示します。
* もし入力された数字が `target` と一致したら、「正解！」と表示して **`break`** でループを抜けてください。
* 一致しなければ「ハズレ！」と表示してループを続行してください。

> **ヒント:** 入力処理が大変な場合は、とりあえず「ループの中で変数をカウントアップし、特定の数字になったら抜ける」という構造から作ってみてもOKです。

---

### 第2問：`loop` から値を返して代入する

Rust特有の「値を返す `loop`」の練習です。

* 変数 `mut counter = 0` を用意します。
* `loop` の中で `counter` を 1 ずつ増やします。
* `counter` が **13** になったとき、**`counter` の値を 10 倍にした数値**を `break` で返してください。
* ループの外側で `let result = loop { ... };` のように受け取り、最後に `result` を表示してください。

---

### 第3問：二重ループと「ループラベル」

多重ループを一気に抜ける「ラベル」の練習です。

* 外側のループに `'outer` というラベルを付けます。
* 内側のループの中で、2つの変数が特定の組み合わせになったら、**外側のループごと**終了させてください。

**期待する動作例:**

```rust
'outer: loop {
    println!("外側ループ開始");
    loop {
        println!("  内側ループ");
        // ここで何か条件が満たされたら、'outer を break する
    }
}
println!("すべて終了");

```

---

## 💡 実装のポイント（再確認）

* **無限ループの防止:** `loop` は必ずどこかに `break` がないとプログラムが止まらなくなります。
* **セミコロンの注意:** `let x = loop { break 10 };` のように値を返す場合、最後の `};` にはセミコロンが必要です。
