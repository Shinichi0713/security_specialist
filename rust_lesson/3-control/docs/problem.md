## 📝 Rust `if`文 練習ドリル

### 第1問：数値の正負判定

変数 `num` に入った数値が、以下の条件に従ってメッセージを表示するプログラムを書いてください。

* 0より大きければ `"Positive"`
* 0より小さければ `"Negative"`
* 0ちょうどであれば "Zero"
  と表示してください。

### 第2問：うるう年判定（論理演算子の練習）

変数 `year` が「うるう年」かどうかを判定してください。

* **条件:**
  1. 400で割り切れるなら、うるう年。
  2. そうでなく、100で割り切れるなら、平年。
  3. そうでなく、4で割り切れるなら、うるう年。
  4. それ以外は、平年。
* **ヒント:** 割り切れるかどうかは、余りを求める `%` 演算子を使います（例: `year % 4 == 0`）。

### 第3問：if式による値の代入

## 💡 解答のヒント

Rustの `if` はカッコ `( )` が不要な代わりに、処理内容を囲む波カッコ `{ }` は必須です。
